# StressDebugger

하루 감정 로그를 분석해서 빡침 원인 포렌식, 반박불가 스트레스 정당화, 싸가지있게 위로까지 자동으로 수행하는 멘탈 디버깅 툴

## 기술 스택

- Flutter (Mobile/Web App)
- Spring Boot 3.2 (API Server)
- Python + Flask (AI Microservice)
- PostgreSQL (Database)
- JavaScript + Chart.js (Dashboard)
- Docker + Docker Compose

## 프로젝트 구조

```
StressDebugger/
├── flutter_app/          # Flutter 앱
├── spring_api/           # Spring Boot API
├── python_service/       # Python AI 서비스
├── js_dashboard/         # JS 대시보드
├── docker-compose.yml    # 도커 컴포즈 설정
└── assets/              # 공통 리소스
```


## 핵심 기능

1. **포렌식 엔진**: 감정 요소 수치화
2. **정당화 엔진**: 반박불가 논리로 기분 합리화
3. **위로 엔진**: 싸가지있게 따뜻한 위로
4. **대시보드**: 히스토리 시각화
